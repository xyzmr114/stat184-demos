# stat184-demos

Demo R scripts for STAT 184 coursework at Penn State.

## Contents

- `data_wrangling_demo.R` — data wrangling with `dplyr` and `tidyr`
- `ggplot_demo.R` — data visualization with `ggplot2`
- `eda_demo.R` — exploratory data analysis (summary stats, correlations)

## Usage

Open any `.R` file in RStudio and run it. Make sure the required packages are installed:

```r
install.packages(c("dplyr", "tidyr", "ggplot2"))
```
